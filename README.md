
           #  README File Generator
           
           ## Description

           An application built using Python and InquirerPy which allows the user to answer a set of questions to generate a markdown file for GitHub. 
            
           ## Installation 

           Git Clone to local machine. 
   
           ## Useage 
           
           Python main.py in the terminal, answer each of the questions, and then once the README file is generated you can move this to your repo folder.
